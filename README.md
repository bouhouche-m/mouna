# Bibliothèque Numérique – Mini Projet POO

## Description
Mini-projet Python utilisant la Programmation Orientée Objet pour gérer une bibliothèque.

## Fonctionnalités
- Ajout de livres
- Ajout d'utilisateurs
- Emprunt de livres
- Retour de livres
- Affichage des livres

## Technologies
- Python 3
- UML (PlantUML)
